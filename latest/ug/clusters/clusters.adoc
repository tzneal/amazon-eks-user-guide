= Managing Amazon EKS clusters
:info_titleabbrev: Clusters

This section provides in-depth guidance on the full lifecycle management of Kubernetes clusters. It covers cluster creation, monitoring cluster health, updating Kubernetes versions, and deleting clusters. It also includes advanced topics on how to configure your cluster, including configuring endpoint access, enabling/disabling Windows support, setting up private clusters, implementing autoscaling, and enhancing resilience with Zonal Shift for traffic redirection in multi-AZ setups.
